Hey there! I am a software developer and here you can find some projects I have done during my studies and in my spare time.

Projects

Master thesis work - Subsurface scattering

Did my master thesis work at Autodesk Stingray where I worked with the rendering team for the game engine. My thesis work was to implement skin shading into their rendering pipeline and simulate the light transport in translucent objects in real-time. The implementation was done in hlsl/glsl.

Procedural generated water

Using C++ and openGL I created procedural generated water and terrain. See a video of the project at youtube . See project on github.

Fur plugin for Maya

A fur plugin created for Maya in C++ using the Maya API. The hair uses a mass spring system. This was done as part of a project in school. See a video of the project at vimeo . See project on github.

Tank shooter

A tank shooting game created with Unity and a leap motion so the tank is controlled with hand gestures. The game logic was written in C#. See project on github.

Monte-carlo Ray tracer

This was a project in the course Advanced global illumination and rendering where I created a monte carlo ray tracer. This was done from scratch in C++. see project on github.

Froggy

Froggy is a game created in a course Design of computer games and is created in Unity3d. See a video of the game at youtube . See project on github.

Genrefy - music genre classifier

A music genre classifier using the machine learning methods K-Nearest Neighbours and Neural Network. The backend was done in python and then presented on a simple webpage. See project on github.

Splash - A game using augmented reality

This is a game created for android using augmented reality and a peer to peer network. The game was created together with a couple of classmates as a part of a course. We used Metaio SDK for the augumented reality parts. See a video of the game at youtube . See project on github.

Guitar tuner - web app

A guitar tuner created as a web app during a course in school together with two classmates. see project on github.

CalcSmart - A smarter calculator

A calculator I created as a hobby project written in C++ using SDL. See project on github.
